# Sentiment-Aspect-Analysis-of-Bank-Reviews-Using-IndoBERT-and-Bi-LSTM
Developed a sentiment aspect-based analysis system for bank customer reviews from Google Maps. The project utilized IndoBERT for feature extraction and Bi-LSTM model. Implemented handled class imbalance using SMOTE/ADASYN. The system also predicted customer ratings and identified the worst service aspects per bank over time.
